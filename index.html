• $ git clone [repositoryURL] - копировать репозиторий в директории, которая сейчас открыта в git bash
  • $ git init [project name] - создать новый локальный репозиторий
  • $ git config --global user.name "[username]" - определяет имя пользователя для поля автора коммитов
  • $ git config --global user.email "[mailbox]" - определяет адрес электронной почты для поля информации о комиттах
  • $ git status - показывает накопившиеся изменения/конфликты
  • $ git pull origin [ветка] - стянуть изменения ветки
  • $ git add [file] - выбрать файлы для пуша, или пробел+точка чтоб выбрать все файлы
  • $ git commit - закоммитить файлы, -m - "комментарий"
  • $ git push - запушить локальные изменения в репозиторий
  • $ git branch - просмотр веток, арумент -d удалит ветку
  • $ git branch [name] - создать новую ветку с именем [name]
  • $ git merge [branch] - cделать слияние указанной ветки в активную
